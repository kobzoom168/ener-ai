3) Agent Specs
🧠 Brain
command:  /note หรือพิมพ์ตรงๆ
input:    ข้อความใดก็ได้
output:   บันทึก + จัดหมวด + extract tasks
หมวด:    task / idea / question / feeling / random
📅 Summary
command:  /today
trigger:  อัตโนมัติ 21:00 ทุกวัน
output:   สรุปวันแยกหมวด + pending + พรุ่งนี้ top 3
format:   แยก section ชัดเจน อ่านง่ายใน Telegram
✅ Task
command:  /task, /done X, /tasks
output:   สร้าง / ดู / ปิด task
fields:   title, priority, deadline_hint, status, tags
📰 News
command:  /news
trigger:  อัตโนมัติ 08:00 ทุกวัน
source:   RSS feeds ที่อยู่ใน whitelist เท่านั้น
output:   top 5 ข่าว + relevance ต่อกบ + action
🔁 Learn (ใหม่)
command:  /learn หรือ /mistake
input:    "วันนี้พลาดเรื่อง X เพราะ Y"
output:   บันทึกใน lessons_learned table
trigger:  weekly review ดึงขึ้นมาเตือน
หลักการ: จำสิ่งที่ผิด → ไม่แนะนำซ้ำ
💡 Brainstorm
command:  /think หรือ /brainstorm
input:    ไอเดียหรือคำถาม
process:  AI A (โจมตี) ↔ AI B (แก้) ↔ AI C (วิเคราะห์)
          วนกัน 3 รอบ
output:   สรุป best version + verdict: go/pivot/stop

4) AI Personality
ชื่อ: ยังไม่ตั้ง (กบตั้งได้)

บุคลิก:
- เชื่อว่าจิตวิญญาณและเทคโนโลยีไปด้วยกันได้
- หาข้อมูลมาสนับสนุนมุมมองกบ ไม่ใช่ขัดแย้ง
- พูดตรง ไม่อ้อมค้อม
- ไม่ disclaimer ไม่ safe mode เรื่องความเชื่อ
- ถ้าไม่รู้ → บอกตรงๆ แล้วไปหาข้อมูล
- จำทุกอย่างที่กบเคยบอก

ภาษา:
- ไทยเป็นหลัก
- ใช้อังกฤษเฉพาะ technical terms

5) Memory Design
3 ชั้น:

SHORT-TERM (session นี้)
└── บทสนทนาล่าสุด 20 messages
└── เก็บใน RAM

WORKING MEMORY (active)
└── open tasks
└── คำถามค้างไว้
└── weekly focus
└── เก็บใน SQLite: memories table

LONG-TERM (ถาวร)
└── belief profile ของกบ
└── lessons learned
└── decision log
└── pattern ของกบ
└── เก็บใน SQLite: long_term_memories table

6) Daily Log Structure
ทุกวัน AI บันทึกแยกหมวดอัตโนมัติ:

📝 NOTES     — สิ่งที่กบจด
✅ TASKS     — งานที่สร้าง/ปิด
💡 IDEAS     — ไอเดียที่เกิด
❌ MISTAKES  — สิ่งที่พลาด
📰 NEWS      — ข่าวที่เกี่ยวข้อง
🔮 INSIGHTS  — สิ่งที่น่าสนใจจากบทสนทนา

→ เก็บใน daily_logs table
→ ดึงขึ้นมาใน /today ได้เลย
→ weekly review ดึงทั้งสัปดาห์มาสรุป
